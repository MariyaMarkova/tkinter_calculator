GUI calclultor 

Simple project which demostrates calculator. User can use math operators such as addition, division, substraction,
multiplcation. Individual can delete whole result by clicking on button C. If person try calculating as number divide 0 it will return message "There is error". Also you can convert calculator.py to exe in order to make app and use it without help of pycharm. Link will be 
bellow after libraries. Also try to use commands to convert .py to .exe in the terminal in pycharm.

Libraries:

- tkinter
